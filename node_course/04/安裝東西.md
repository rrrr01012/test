## mongod 使用
* 當使用mongodb 會默認在使用指令的硬碟中的一個/data/db 所以要先設
* mongod 開啟
* crtl +  c 關閉
* 連接 指令 mongo
* 退出 再連接狀態輸入exit

## mongod 基本命令
* show dbs  
    + 查看顯示資料庫
* db
    + 查看當前資料庫
* use + 資料庫名稱
  + 切換到指定資料(如果沒有會新建)
* 插入資料\
